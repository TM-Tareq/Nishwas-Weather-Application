## 📖 Overview

Nishwas is a comprehensive weather intelligence platform designed for the people of Bangladesh. It goes beyond conventional weather forecasting by integrating real-time air quality data with personalized health insights, smart route planning, and community-driven environmental reporting.

The platform empowers users to make informed daily decisions — whether it's choosing the safest route through the city, planning outdoor activities based on individual health conditions, or contributing to a collective understanding of local environmental conditions.

## ✨ Key Features

### 🌤️ Hyperlocal Weather Intelligence
Real-time weather and air quality data for all 64 districts of Bangladesh, with GPS-based hyperlocal forecasts and interactive parameter overlays.

### 🏥 Health-Aware Suggestions
Personalized outdoor activity recommendations based on user health profiles — asthma, diabetes, age group, and allergies — combined with current AQI, UV index, and weather conditions.

### 🛣️ Smart Route Planning
Multi-criteria route comparison offering three options: **Fastest** (time-optimized), **Healthiest** (low AQI exposure), and **Coolest** (low heat stress) — ideal for sensitive groups and outdoor workers.

### 🗺️ Interactive Environmental Map
Live map visualization with parameter overlays (AQI, temperature, humidity, rainfall, wind), click-to-explore district details, and animated weather patterns.

### 🏆 Regional Insights
Division and district-level leaderboards across multiple environmental parameters, with side-by-side comparison tools for any two locations.

### 👥 Community-Driven Reports
Citizen-powered weather reports with photo verification, upvoting, and community-based authentication — bringing ground-truth data to the platform.

### 🏅 Engagement Through Gamification
Points, badges, streaks, and contributor leaderboards reward consistent engagement and quality community contributions.

### 📔 Personal Weather Diary
Long-term personal tracking with mood and health correlation insights — discover how weather patterns affect your well-being over time.

### 🌐 Bilingual Experience
Fully bilingual interface (English + বাংলা) with native typography support, designed for inclusivity.

### 📱 Progressive Web App
Installable, offline-capable, mobile-first experience that works seamlessly across devices.

## 🛠️ Tech Stack

### Frontend
- **React 18** with Vite for blazing-fast development
- **Tailwind CSS** for utility-first styling
- **TanStack Query** for server state management
- **Zustand** for client-side global state
- **React Hook Form + Zod** for performant, type-safe forms
- **Leaflet + OpenStreetMap** for interactive mapping
- **Recharts** for data visualization
- **i18next** for internationalization

### Backend
- **Spring Boot 3** with **Java 17 LTS**
- **Spring Security + JWT** for stateless authentication
- **Spring Data JPA + Hibernate** for data persistence
- **Flyway** for versioned database migrations
- **MySQL 8** as the primary data store

### External Integrations
- **Open-Meteo API** — Weather and air quality data
- **OpenRouteService** — Multi-modal routing
- **Cloudinary** — Image storage and optimization

## 🚀 Getting Started

### Prerequisites

Make sure the following are installed on your system:

- Node.js 20.x LTS or higher
- Java 25 LTS
- MySQL 8.x
- Git

### Installation

Clone the repository:

\`\`\`bash
git clone https://github.com/TM-Tareq/Nishwas-Weather-Application.git
cd Nishwas-Weather-Application
\`\`\`

#### Frontend

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

The application will be available at `http://localhost:5173`.
